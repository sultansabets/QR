# Система проверки сертификатов (MVP)

Веб-приложение для автоматической проверки сертификатов поверки средств измерения (СИ) в метрологической лаборатории (Казахстан).

## Проверки

1. Наличие типа СИ в реестре ГСИ РК (Государственная система обеспечения единства измерений).
2. Соответствие области аккредитации лаборатории.

## Стек

- **Frontend:** React, TypeScript, Tailwind CSS, Vite
- **QR:** html5-qrcode
- **PDF:** pdfjs-dist
- **Отчёты:** jspdf, jspdf-autotable
- **Маршрутизация:** react-router-dom
- **Данные:** JSON (mock)

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## Режимы ввода

- **QR-код** — сканирование QR с сертификата
- **PDF** — загрузка PDF (извлечение текста)
- **Вручную** — ввод данных формы

## Структура

- `src/components/` — UI-компоненты
- `src/pages/` — страницы (главная, проверка, история, результат)
- `src/services/` — проверка, история, отчёт
- `src/mock-data/` — реестр ГСИ РК, область аккредитации, примеры сертификатов

Весь интерфейс на **русском языке**.
